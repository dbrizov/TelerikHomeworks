Main features

  • Adding, modifying and deleteing of posts
  • Organizing posts via automated tag system
  • Adding comments to posts, restricted to logged users

Client Side

A Kendo UI MVVM application, using bootstrap css framework

Admin Side

An angularJS MVC application, using bootstrap css framework with custom admin template
